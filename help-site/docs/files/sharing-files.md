# Sharing Files

You can share files and folders with your staff or with clients outside your business.

---

## Share with a staff member (internal sharing)

If a staff member has a Nextcloud account, you can share directly with them:

1. Log in to `https://files.YOUR_DOMAIN.woodhead.tech`
2. Right-click the file or folder → **Share**
3. In the **Search for users** field, type their name or email
4. Select them from the dropdown
5. Choose their permission level:
   - **Can view** — they can see and download but not edit
   - **Can edit** — they can add, edit, and delete files in the folder
6. Click **Share**

They'll get a notification and the shared folder will appear in their Nextcloud.

---

## Share with a client (external link)

You can create a shareable link that anyone can use — no Nextcloud account required.

1. Right-click the file or folder → **Share**
2. Click **+ New share link**
3. Options:
   - **View only** — they can see and download but not upload
   - **Upload only (File drop)** — they can upload files but can't see what's there (useful for receiving documents from clients)
   - **View and upload** — full access to view and upload
4. Set an expiry date (recommended — don't leave client links open forever)
5. Optionally set a password for the link
6. Click **Copy link** and send it to your client

---

## Stop sharing

To remove access:

1. Right-click the file or folder → **Share**
2. Click the X next to the person or link you want to remove
3. Their access is removed immediately

---

## See who has access to a file

Right-click the file → **Share** — the sharing panel shows everyone who has access
and what level of permission they have.

---

## Tips

- Share **folders** rather than individual files when you're collaborating on a project — it's easier to manage
- Set expiry dates on client links so they don't stay open indefinitely
- Use **File drop** when you want clients to send you documents without seeing anything else
