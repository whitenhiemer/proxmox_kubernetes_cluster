# File Storage

Your ShopStack includes Nextcloud — a file storage and sharing system for your business.
Think of it as your own private Dropbox or Google Drive, running on your own system.

**Your file storage:** `https://files.YOUR_DOMAIN.woodhead.tech`

Log in with your Nextcloud username and password (from the handoff email).

---

## What you can do

- Store and access files from any device (computer, phone, tablet)
- Share files and folders with staff or clients
- Sync your desktop so files appear in a folder on your computer
- Access files from your phone with the Nextcloud mobile app
- See who made changes and when

---

## Using Nextcloud in your browser

The simplest way to use Nextcloud is through your browser — no software installation needed.

1. Go to `https://files.YOUR_DOMAIN.woodhead.tech`
2. Log in with your credentials
3. Drag and drop files to upload, click folders to navigate, right-click for options

**To upload a file:**
- Drag it from your desktop into the browser window, or
- Click the **+** button → **Upload file**

**To create a folder:**
- Click **+** → **New folder** → type a name → press Enter

**To share a file or folder with someone outside your business:**
- Right-click the item → **Share**
- Turn on **Share link** — copy the link and send it to them
- Set an expiry date if you don't want the link to work forever
