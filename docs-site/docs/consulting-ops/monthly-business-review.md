# Monthly Business Review Runbook

Human-executable. No AI required. Run this on the last Friday of every month.
Takes 30–45 minutes.

**Key docs:**
- V2MOM: `~/WOODHEAD_CONSULTING/v2mom-2026.md`
- Budget roadmap: `~/WOODHEAD_CONSULTING/budget-roadmap.md`
- Sales pipeline: `~/WOODHEAD_CONSULTING/sales-action-plan.md`
- LinkedIn schedule: `~/WOODHEAD_CONSULTING/linkedin/schedule.md`
- Backlog: `~/.claude/skills/consulting-business/references/backlog.md`

---

## Step 1 — Revenue (10 min)

Open Stripe dashboard → Payments → filter by current month.

Record:

| Source | This month | Last month | Delta |
|--------|-----------|-----------|-------|
| Engineering consulting | $ | $ | |
| ShopStack setup fees | $ | $ | |
| ShopStack monthly managed | $ | $ | |
| ShopStack Online | $ | $ | |
| **Total** | **$** | **$** | |

Compare against the Year 1 projection in `budget-roadmap.md` — are you ahead,
on track, or behind?

**If behind:** Is it a pipeline problem (not enough conversations) or a conversion
problem (conversations aren't closing)? The answer determines where to focus next month.

**AWS costs (Cloud tier clients):** Check AWS billing console — EC2 costs should
be ~$50–60/mo per cloud client. Flag if any instance is running unusually high.

## Step 3 — Pipeline (10 min)

Open `sales-action-plan.md` and review both tracking tables.

**Engineering consulting:**
- How many active conversations this month?
- How many are at "Discovery Call" stage or beyond?
- Any proposals outstanding? Follow up if >7 days with no response.
- Any "not now" responses to add to the 90-day revisit list?

**ShopStack:**
- Any new leads from Facebook, LinkedIn, or referrals?
- Free setup slots: still 2 open? Update the pinned Facebook post if slots change.
- Any active free setups in progress?

**Combined health check:**

| Metric | This month | Target | On track? |
|--------|-----------|--------|-----------|
| Warm network conversations (cumulative) | | 20 by Jun 1 | |
| LinkedIn ICP connections | | 500 by Jul 2026 | |
| ShopStack active clients | | 5 by Q4 2026 | |
| Advisory retainers active | | 2 by Q1 2027 | |
